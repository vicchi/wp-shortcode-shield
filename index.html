WP Shortcode Shield

WP Shortcode Shield is a WordPress plugin that allows posts and pages to easily document WordPress shortcodes without the shortcode being expanded.

TThis plugin allows you to refer to a WordPress shortcode within the content of posts and pages without that shortcode being automagically expanded by WordPress. The plugin takes advantage of the fact that the WordPress ShortCode API does not support nested shortcodes to allow this plugin's shortcode to wrap the shortcode being documented.

Authors

WP Shortcode Shield is written, produced and hacked by Gary Gale.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/vicchi/wp-shortcode-shield

WordPress Installation

You can install this plugin from the Plugins Dashboard on a self hosted Wordpress.org install or on WordPress.com, or from the official WordPress plugins repository.

Get the source code on GitHub : vicchi/wp-shortcode-shield
